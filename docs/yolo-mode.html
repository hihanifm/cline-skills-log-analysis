Key Building Blocks

🔡

Problem Embedding

Embed the incoming problem description (and optionally a few sampled log lines) to find semantically similar past issues in the knowledge DB — seeds the initial template shortlist.

semantic-search
📐

Template Ranker

Scores each candidate template by two signals: semantic relevance to the current problem, and historical success rate on similar past problems. Returns a ranked shortlist.

retrieval + scoring
🎯

Confidence Scorer

After each synthesis pass, evaluates the generated report: Did the LLM identify a root cause? Are key patterns covered? Returns a confidence score that drives the loop exit.

LLM + heuristics
🗄️

Knowledge DB

Grows with every completed run. Stores: problem description → templates used → RCA outcome → confidence. Future runs use this to bootstrap a better template shortlist from day one.

persistent store
